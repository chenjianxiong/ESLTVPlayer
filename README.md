# XiaoMi TV Video Player

A lightweight, feature-rich video player optimized for XiaoMi TV with custom navigation controls, overlay capabilities, and resume playback functionality.

## Features

- **Video Format Support**: MP4, MKV, AVI, MOV, WebM, FLV
- **Custom Navigation**: Configurable seek times (1-60 seconds)
- **Resume Playback**: Automatically resume from last position
- **Overlay Feature**: Customizable overlay for testing and debugging
- **File Browser**: Easy navigation through internal and external storage
- **Settings GUI**: Full configuration for all features

## System Requirements

- **Device**: XiaoMi TV
- **Processor**: Cortex A55 (4 cores)
- **OS**: MiTV OS 2.9.6 or higher
- **Android Version**: Android TV 9.0+

## Controls

- **Left/Right**: Seek backward/forward (configurable)
- **Up/Down**: Show/hide overlay
- **Center/OK**: Play/Pause
- **Back/Home**: Exit and save position
- **Menu**: Open settings

## Building

1. Clone the repository
2. Open in Android Studio
3. Build and run on XiaoMi TV or Android TV emulator

## Development

- **Language**: Kotlin
- **Media Framework**: ExoPlayer 2.x
- **UI Framework**: Android TV Leanback Library
- **Database**: Room (SQLite)
- **Build System**: Gradle
